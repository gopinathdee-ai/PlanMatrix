## Quick Start from Zip

### 1. Extract
```bash
unzip floorplan-system-boilerplate.zip
cd floorplan-system-boilerplate
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure Environment
Edit `.env.local`:
```env
DATABASE_URL="Server=localhost;Database=floorplan_db;User Id=sa;Password=YourPassword123;Encrypt=false;"
NEXTAUTH_SECRET="$(openssl rand -base64 32)"
NEXTAUTH_URL="http://localhost:3000"
ADMIN_USERNAME="admin"
ADMIN_PASSWORD="Password123"
PDF_STORAGE_PATH="./public/floor-plans"
```

### 4. Setup Database
```bash
npm run db:generate   # Generate migrations
npm run db:push       # Push to SQL Server
```

### 5. Start Dev Server
```bash
npm run dev
```

Visit http://localhost:3000 and login with `admin` / `Password123`

### 6. Build with Claude Code
Paste the contents of `CLAUDE_CODE_PROMPT_FINAL.md` into Claude Code to complete the implementation.
